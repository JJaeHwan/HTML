필터로 사용
:enabled ==> 사용 가능한 요소를 걸러내기(선택하기)
:disabled ==> 사용 불가능한 요소를 걸러내기(선택하기)

속성으로 사용
enabled ==> 사용할 수 없는 속성- 속성으로 사용할 수 없다.
disabled ==> 사용 불가능한 요소의 속성을 설정한다.









필터로 사용
:checked ==> checkbox 또는 radio에서 선택한 요소를 걸러내기(선택하기)

속성으로 사용
checked="checked"" ==> checkbox 또는 radio 실행시 초기값 설정


취 미 : 여행 누워있기 메이플 공부 유튜브시청

필터로 사용
:selected : select-optioned에서 선택한 요소를 걸러내기(선택하기)

속성으로 사용
selected : select -option 또는 radio 실행시 초기값 설정
